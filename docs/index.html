How Points Works

🎥 Depth Source → 🧹 Cleanup → 🎨 FX Pipeline → 📌 Pinout GPU → 🖥 Render 60fps

Points never leave the GPU. The renderer fetches directly from the unprojector's interleaved 24-byte buffer — no CPU readback, no frame drops at 200,000 points.

Quick Example: Depth → Palette → Output

Minimal Patch

graph LR depth[Depth
mode: metric] -->|depth| palette[Palette
map: thermal] palette -->|color| out[Output] depth -->|position| pinout[Pinout] depth -->|z| pinout pinout -->|position| out pinout -->|z| out

Wire Depth → Pinout → Output for the point cloud. Add Palette between Depth and Output for color. Done — you have a thermal-colored LiDAR cloud.

Getting Started

1. Add a Depth Node

This reads your device's LiDAR, TrueDepth, or RGB camera and turns it into per-point data.

2. Wire to Pinout

Pinout snaps every point to its grid home and pushes Z by the depth value — the classic pin-screen toy.

3. Connect to Output

Output is the final render sink. Everything routes through it to appear on screen.
