Points

A live point cloud instrument. Wire 110 GPU nodes together. Modulate with audio, MIDI, OSC, and gestures. Create real-time Metal-rendered visuals from LiDAR, TrueDepth, or RGB camera sources at 200,000 points, 60fps.

How It Works

graph LR A[Depth Source
LiDAR / TrueDepth / RGB] --> B[Cleanup Pipeline
EMA / Fill / Cull] B --> C[FX Nodes
Move / Color / Shape / Signal] C --> D[Pinout
GPU Point Cloud] D --> E[Output
Metal Render 60fps] C --> F[External I/O
NDI / Record]

Points never leave the GPU. The renderer fetches directly from a 24-byte interleaved buffer. No CPU readback. No frame drops at 200,000 points.

Camera Modes

ModeSensorFPSDescription
CLOUDBack LiDAR30True metric 3D depth. Objects keep real size at any distance.
FACEFront TrueDepth30Face mesh + 52 blendshape modulation sources.
LUMENBack RGB60Luma-as-depth at full 60fps. Grid-stable pins for per-pin history FX.

Getting Started

Interface

ZoneLocationPurpose
ViewportTopLive point cloud. 200,000 points at 60fps via Metal.
NodeviewBottom barDrag nodes from the palette. Wire them together.
CameraviewBottom barSliders, pads, and buttons for the selected node.

Navigating the Nodeview

1. Tap + to open the node palette. Nodes are organized by family.

2. Tap a node to add it to the canvas.

3. Drag from an output port (right side) to an input port (left side) to wire.

4. Tap any node to select it; its parameters appear in the Cameraview.

Navigating the Cameraview

Sliders adjust float parameters in real time. Changes apply instantly to the GPU.

Expose tap the expose icon on any slider to create an input port. Wire a signal in to drive it.
