MIDI

Connect any MIDI controller via USB or Bluetooth. Points auto-detects devices.

Audio Analysis

Real-time microphone FFT via Accelerate. Mic activates only when audio nodes are in the graph.

NodeOutputRange
Audio Volumevolume (signal)0-1 RMS
Audio FFTlow/mid/high (signal)0-1 per band

The modulation chain: Audio -> Threshold -> Envelope -> Curve -> Spring -> exposed param.
