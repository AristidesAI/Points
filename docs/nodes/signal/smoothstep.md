# Smoothstep

**ID** `smoothstep` · **Family** SIGNAL · **GPU** (interpreterOp)

Hermite interpolation between edge0 and edge1.

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `edge0` | −2 – 2 | 0 | Start edge |
| `edge1` | −2 – 2 | 1 | End edge |

| Port | Direction | Type |
|------|-----------|------|
| `in` | input | fieldFloat |
| `out` | output | fieldFloat |
