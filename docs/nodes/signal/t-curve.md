# Curve

**ID** `t-curve` · **Family** SIGNAL · **CPU** (control)

Reshapes a 0–1 value through an easing curve.

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `curve` | quadIn/Out/InOut, cubicIn/Out/InOut, backIn/Out/InOut, bounceOut, elasticOut | quadOut | Easing curve |

| Port | Direction | Type |
|------|-----------|------|
| `in` | input | signal |
| `out` | output | signal |
