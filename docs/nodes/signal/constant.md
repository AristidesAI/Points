# Constant

**ID** `constant` · **Family** SIGNAL · **GPU** (interpreterOp)

Outputs a fixed value to every pin.

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `value` | −4 – 4 | 1 | Constant output |

| Port | Direction | Type |
|------|-----------|------|
| `out` | output | fieldFloat |
