# Screen Blend

**ID** `color-screen` · **Family** COLOR · **GPU** (interpreterOp)

Screen blend: 1−(1−a)(1−b). Only ever brightens.

| Port | Direction | Type |
|------|-----------|------|
| `a` | input | fieldColor |
| `b` | input | fieldColor |
| `out` | output | fieldColor |
