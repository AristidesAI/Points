# Color Invert

**ID** `color-invert` · **Family** COLOR · **GPU** (interpreterOp)

Photographic negative with crossfade control.

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `amount` | 0 – 1 | 1 | 0=original; 1=full negative |

| Port | Direction | Type |
|------|-----------|------|
| `color` | input | fieldColor |
| `out` | output | fieldColor |
