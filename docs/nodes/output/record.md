# Record

**ID** `record` · **Family** OUTPUT · **Render** (render-read)

Records viewport to H.264 MP4 in Photos library.

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `fps` | 30 / 60 | 30 | Recording fps |
| `quality` | low / medium / high | high | Encoding quality |
