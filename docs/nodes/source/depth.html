Depth

SOURCE GPU. The primary sensor. Reads depth camera and produces per-pin nearness, 3D position offset, and Z push.

How It Works

The Depth node sits at the boundary between the physical world and the GPU. Every frame, the depth sensor produces a texture where each pixel holds a distance in metres. The node computes three outputs:

1. Nearness (depth output): (far - raw) / (far - near). 1 = at NEAR, 0 = at FAR. Holes = 0. This is your modulation signal — wire to Palette.t, Size, Gate thresholds.

2. Position offset (position output): 3D vector per pin. METRIC mode uses real camera intrinsics — objects keep true size. FREE mode uses simpler fan projection.

3. Z push (z output): Signed scalar = (FOCUS - raw) x SEPARATION. Positive = in front of wall. Negative = behind. This is geometry — wire to Output.z or through Lag/Trail/Spring.

Common Mistakes

Wire depth into Output.z: scene compresses flat. Wire z into Palette: meaningless colors (signed, unbounded). Rule: depth for effects, z for geometry.
