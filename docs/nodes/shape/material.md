# Material

**ID** `material` · **Family** SHAPE · **Render** (render-read)

How pins are shaded when lit by Light nodes.

## Parameters

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `shading` | unlit / lit / matcap | lit | Shading model |
| `roughness` | 0 – 1 | 0.5 | Surface roughness |
| `metallic` | 0 – 1 | 0 | Metallic reflection |
