# Stem

**ID** `stem` · **Family** SHAPE · **Render** (render-read)

Styles the arms from the Z-wall to each cap. Turn ARMS on in Depth to see them.

## Parameters

| Param | Range | Default | Description |
|-------|-------|---------|-------------|
| `profile` | square / round / blade | square | Arm cross-section |
| `thickness` | 0 – 1 | 0.3 | Arm width |
| `taper` | 0 – 1 | 0.2 | Narrowing toward cap |
